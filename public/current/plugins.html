Add an Official Plugin

You can load any of these official plugins. We believe these plugins are free of privacy leaks, although bugs are possible and we make no guarantees. You can also load unofficial plugins, see below. You can either fetch the plugin over Freenet or from our central servers:

Fetch over Freenet. This is untraceable, safe, and may be slow.
Fetch over the web from our central servers. WARNING: This will give away the fact that you are running Freenet, and won't work if our website is blocked! It may even be possible for a wiretapper to tell what you are trying to load, and thus connect it with when e.g. a Freemail identity was created.

Load an official plugin:

Communication Plugins
FlogHelper (version 29) - Easy to use tool for creating a blog on Freenet. Requires WebOfTrust.
Freemail (version 15) - Email over Freenet. Requires an email client (e.g. Thunderbird).
Freetalk (version 13) - Experimental anonymous forum. Currently has severe performance and reliability problems. Requires WebOfTrust.
Connectivity Plugins
MDNSDiscovery (version 2) - Announces Freenet on your local network (LAN) so that clients can automatically connect to it. Not used much at the moment.
SNMP - Enables you to remotely monitor your Freenet node's status via tools such as MRTG.
Example Plugins
HelloFCP - FCP test plugin. Does nothing useful, only for developers of new plugins.
HelloWorld - Test plugin. Does nothing useful, only for developers of new plugins.
TestGallery (experimental) (version 1) - Simple gallery page plugin, not currently used.
Indexing Plugins
Freereader (version 4) - Tool to insert RSS and ATOM feeds as freesites.
Spider (version 50) - Tool to create and automatically upload new format freesite search indexes, by automatically following links from your bookmarks. Causes very heavy load on your Freenet node, don't use it unless you have a powerful computer! Requires that the Library plugin also be loaded.
XMLLibrarian (deprecated) (version 26) - Obsolete tool to use old format freesite search indexes. See Library.
XMLSpider (deprecated) (version 48) - Obsolete tool to generate old format freesite search indexes. See Spider.
