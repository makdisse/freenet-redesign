Upload a freesite

You can anonymously upload a web site to Freenet ("insert a Freesite"). Once it has been uploaded, it will remain on Freenet for as long as people continue to access it occasionally. It is distributed across the network and therefore also available if your computer is turned off. And the most important fact: If the security level of Freenet has been configured properly it should be very difficult to find out who has uploaded a Freesite.

If you just want to create a simple blog, we recommend you use FlogHelper. You can load this on the plugins page, and then use it from the menu.

If you want to upload a website you have mirrored from somewhere, or something larger or more complex, you will need a tool called jSite. Get it from from our web site, or you can get version 0.6.2 from Freenet (more information).

Once you have downloaded jSite, and constructed your website using an HTML editor such as KompoZer (free), work through one of the following tutorials which are hosted on Freenet:

  • Publish! - A short tutorial on how to insert freesites
  • Freesite HOWTO - Another, longer tutorial on how to insert freesites

Alternatively, Thingamablog is a user-friendly blog generator, you can download a Freenet-compatible version here or over Freenet.
