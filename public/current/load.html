Freenet Node Configuration
Load management
  • Thread limit Freenet will try to limit its thread usage to the specified value by refusing new requests when it is using too many threads.
  • AggressiveGC modificator Enables the user to tweak the time in between GC and forced finalization. SHOULD NOT BE CHANGED unless you know what you're doing! -1 means: disable forced call to System.gc() and System.runFinalization()
  • Enable the Memory checkEnable the memory check (writes a message in logfile, mandatory for aggressiveGCModificator to have any effect!)
  • Treat local requests as remote requests for bandwidth liability limiting?Enable this for significantly lower bandwidth usage and slightly more security against request starting timing attacks (mostly you don't need to worry about them as correlation attacks are probably easier)
  • Maximum ping time? Freenet will reject all requests if the average ping time to its peers goes above this level. Ping times have proven a reliable indicator of both system and network load, but if your internet connection is unusually slow, you should increase this. Measured in milliseconds.
  • Start rejecting requests at what ping time? Freenet will start to reject some requests when the average ping time to its peers goes beyond this time. Increase it if your internet connection is unusually slow. Measured in milliseconds.
  • Enable new load management (for realtime requests)? (EXPERIMENTAL!)Turn this on to test the new load management code for realtime requests. This is highly experimental and may severely break your node or dramatically reduce or increase its performance! Good luck, and tell us what happens!
  • Enable new load management (for bulk requests)? (EXPERIMENTAL!)Turn this on to test the new load management code for bulk requests. This is highly experimental and may severely break your node or dramatically reduce or increase its performance! Good luck, and tell us what happens!
  • File to store node statistics in File to store node statistics in (not client statistics, and these are used to decide whether to accept requests so please don't delete)
